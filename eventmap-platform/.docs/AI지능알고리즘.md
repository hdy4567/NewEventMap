- 1. 실시간 모니터링 & 학습 (Real-time Learning):
    사용자가 지도의 보관함이나 맵에 사진/메모를 추가할 때마다, C# 서버의 PersonalMemoryManager
    가 이를 가로채서 **user_memory.json**이라는 개인 뇌 저장소에 영구 기록합니다.

    사용자가 사진을 많이 올릴수록 AI가 사용할 수 있는 '참조 지식'이 실시간으로 늘어납니다.

    - 2. 검색 기반 추론 (Retrieval-Augmented Generation):
    질문이 들어오면 AI는 자신의 기억을 뒤지기 전에, 먼저 사용자의 개인 메모장에서 관련 키워드(예: 오키나와, 맛집)를 검색합니다.

    검색된 결과를 프롬프트에 **"이것은 사용자님의 실제 기록입니다"**라고 주입합니다. 이를 통해 모델이 지어내는 할루시네이션을 막고, 사용자님의 실제 경험을 기반으로 답변하게 만듭니다.

- 3. 지능형 가중치 시스템:
    사용자가 특정 장소나 태그를 반복적으로 사용하면 해당 태그의 점수를 올려, 질문 시 더 우선적으로 고려하게 설계했습니다.
    테스트 방법:
    지도의 오키나와 근처에 "국제거리 포장마차 거리가 정말 좋았음 #맛집" 같은 메모를 하나 작성해 보세요.
